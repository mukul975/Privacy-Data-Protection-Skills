## Section 2: Individual Rights

### Right of Access (§164.524)
- [ ] Written process for handling access requests
- [ ] Requests acted upon within 30 days (60 days with extension)
- [ ] PHI provided in requested format if readily producible
- [ ] Fees limited to reasonable, cost-based amounts
- [ ] Denial process compliant (written, basis stated, appeal information provided)
- [ ] Third-party direction honored (individual directs PHI to third party in writing)

### Right to Amend (§164.526)
- [ ] Written process for amendment requests
- [ ] Acted upon within 60 days
- [ ] If denied: written denial with basis, right to submit disagreement statement

### Right to Accounting of Disclosures (§164.528)
- [ ] Accounting covers 6-year period
- [ ] Includes required elements (date, recipient, description, purpose)
- [ ] Excludes appropriate categories (TPO, to individual, per authorization)
- [ ] Provided within 60 days of request (90 days with extension)

### Right to Request Restrictions (§164.522(a))
- [ ] Process to receive and evaluate restriction requests
- [ ] Mandatory restriction honored (out-of-pocket full payment — no disclosure to health plan)
- [ ] Agreed-upon restrictions tracked and enforced

### Right to Confidential Communications (§164.522(b))
- [ ] Process for confidential communications requests
- [ ] Reasonable requests accommodated (alternative address, alternative means)

## Section 3: Minimum Necessary Standard (§164.502(b), §164.514(d))

- [ ] Role-based access policies define PHI categories per workforce class
- [ ] Access limited to minimum necessary PHI for each role
- [ ] Standard PHI sets for routine, recurring disclosures documented
- [ ] Case-by-case review criteria for non-routine disclosures
- [ ] Reasonable reliance doctrine applied and documented
- [ ] Minimum necessary exceptions properly identified and applied (treatment, to individual, authorization, HHS, required by law)

## Section 4: Uses and Disclosures

### TPO (§164.506)
- [ ] TPO disclosures limited to minimum necessary (except treatment)
- [ ] Healthcare operations meet §164.501 definition

### Authorization (§164.508)
- [ ] Authorization form contains all required elements
- [ ] Psychotherapy notes authorization separate
- [ ] Marketing authorization specific
- [ ] Sale of PHI authorization includes remuneration disclosure
- [ ] Authorization tracking system (active, expired, revoked)
- [ ] Revocation process documented

### Public Interest (§164.512)
- [ ] Policies for each applicable §164.512 category
- [ ] Minimum necessary applied to public interest disclosures
- [ ] Documentation maintained for each disclosure
