## 5. CCTV Compliance (EDPB Guidelines 3/2019)

- [ ] Lawful basis documented
- [ ] Signage at boundary of monitored area
- [ ] Signage includes: camera pictogram, controller identity, purpose, reference to full notice
- [ ] Full privacy notice accessible (QR code / website / reception)
- [ ] Retention default <=72 hours (or justified if longer)
- [ ] Access controls for footage viewing documented
- [ ] Privacy masking for non-relevant areas
- [ ] Facial recognition: explicit consent or substantial public interest basis
