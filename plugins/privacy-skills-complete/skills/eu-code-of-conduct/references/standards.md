# EU Code of Conduct Standards Reference

## Primary Legal Provisions

### GDPR Articles 40-41
- **Art. 40**: Codes of conduct — encouragement, content, approval, registration, monitoring
- **Art. 41**: Monitoring of approved codes of conduct — monitoring body requirements, accreditation
- **Art. 40(2)(a)-(k)**: Processing areas codes may cover
- **Art. 40(3)**: Codes as transfer safeguards under Art. 46(2)(e)
- **Art. 40(5)-(7)**: National and transnational approval procedures
- **Art. 40(9)-(11)**: Commission implementing acts, publication, registration

### EDPB Guidelines 1/2019
- **Title**: Guidelines 1/2019 on Codes of Conduct and Monitoring Bodies under Regulation 2016/679
- **Adopted**: June 4, 2019 (version 2.0)
- **Scope**: Detailed guidance on code content, approval process, monitoring body accreditation, and adherence mechanisms

## Regulatory Benefits of Code Adherence

| GDPR Article | Benefit |
|-------------|---------|
| Art. 24(3) | Element to demonstrate controller compliance |
| Art. 28(5) | Element demonstrating processor sufficient guarantees |
| Art. 32(3) | Element to demonstrate security obligation compliance |
| Art. 35(8) | Factor in DPIA impact assessment |
| Art. 46(2)(e) | Appropriate safeguard for international transfers |
| Art. 83(2)(j) | Mitigating factor for administrative fines |

## Approved Codes of Conduct

### EU Cloud Code of Conduct (SCOPE Europe)
- **Approval**: May 2021, Belgian DPA (lead SA), EDPB positive opinion
- **Monitoring Body**: SCOPE Europe AISBL
- **Scope**: Cloud IaaS, PaaS, SaaS processors

### CISPE Data Protection Code of Conduct
- **Approval**: 2023, CNIL
- **Scope**: Cloud infrastructure service providers
