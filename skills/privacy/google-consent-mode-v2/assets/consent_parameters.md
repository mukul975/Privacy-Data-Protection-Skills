# Consent Mode v2 Parameter Reference

| Parameter | Description | EEA Default | US Default |
|-----------|-------------|-------------|------------|
| `ad_storage` | Controls advertising cookies (e.g., _gcl_*, IDE) | denied | granted |
| `ad_user_data` | Controls sending user data to Google for advertising | denied | granted |
| `ad_personalization` | Controls ad personalization and remarketing | denied | granted |
| `analytics_storage` | Controls analytics cookies (e.g., _ga, _gid) | denied | granted |
| `functionality_storage` | Controls functionality cookies (e.g., language) | granted | granted |
| `personalization_storage` | Controls personalization cookies | denied | granted |
| `security_storage` | Controls security cookies (e.g., authentication) | granted | granted |
