# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-15

### Added
- Initial release with 282 privacy and data protection skills
- 20 compliance domains covering GDPR, CCPA/CPRA, HIPAA, LGPD, PIPL, DPDP Act, and 10+ global regulations
- agentskills.io compatible SKILL.md format with YAML frontmatter
- 5-file skill structure: SKILL.md, references/standards.md, references/workflows.md, scripts/process.py, assets/template.md
- Claude Code Plugin Marketplace with 21 installable plugins (1 complete + 20 domain-specific)
- Working Python 3 automation scripts using only standard library for each skill
- Filled real-world templates with realistic organizational data
- Real regulatory citations with enforcement precedents including case names and fine amounts
- GitHub issue templates for no-code contributions from privacy professionals
- Pull request template with compliance verification checklist
- Security vulnerability reporting policy
- Apache 2.0 license

### Regulatory Coverage
- EU GDPR: 50+ skills covering Articles 5-99, EDPB guidelines, CJEU case law
- CCPA/CPRA and 13 US state privacy laws (VCDPA, CPA, CTDPA, TDPSA, OCPA, MTDPA, and more)
- HIPAA Privacy Rule, Security Rule, and Breach Notification Rule
- Brazil LGPD, China PIPL, India DPDP Act, Japan APPI, South Korea PIPA
- Singapore PDPA, Thailand PDPA, Australia Privacy Act, South Africa POPIA
- EU AI Act privacy requirements ahead of August 2026 full enforcement
- Cross-border frameworks: SCCs, BCRs, APEC CBPR, EU-US Data Privacy Framework
- ePrivacy Directive, COPPA, UK AADC, NIST Privacy Framework, ISO 27701
